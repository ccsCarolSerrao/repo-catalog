# How to Run Locally?

1. Clone the repository
    ```bash
    $ git clone git@github.com:olxbr/maturity-level.git
    $ cd maturity-level
    ```


2. Install all dependencies
    ```bash
    $ yarn
    ```


3. Create a _.env_ file based on _.env.example_ file


4. Start the application
    ```bash
    $ yarn start
    ```
